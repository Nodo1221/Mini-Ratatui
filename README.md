# mini-ratatui

A minimal showcase of modern [ratatui](https://ratatui.rs/) (0.30) covering the core concepts: layouts, flex, widgets (stateless and stateful), styling, event handling, all in a single interactive annotated demo. Use for reference.

## Regarding AI
Some AI models tend to generate code for older versions of ratatui (and the documentation can be ambiguous, which hinders lookup), resulting in deprecated / legacy or outright invalid code. I've created the [`SKILL.md`](./SKILL.md) file to address this at least in part, in a similar minimal fashion.

## Run with

```bash
cargo run
```
